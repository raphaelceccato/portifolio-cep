Sobre a página

Esta página faz parte do meu portifólio.

Consiste em uma demonstração de uso de uma API REST simples (neste caso a do ViaCEP), além de também demonstrar o uso de Bootstrap na página.

Uma requisição GET é enviada ao ViaCEP através da função fetch(). A API devolve uma resposta em JSON, que é interpretada como um objeto. Juntamente com ela são usadas Promises e os operadores async/await para processar a resposta de forma assíncrona.
